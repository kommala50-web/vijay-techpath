JavaScript Full Roadmap

Learn JavaScript from basics to advanced concepts like DOM, async programming, APIs, and performance optimization. Click Show Code to view examples.

1. Introduction to JavaScript

Beginner

JavaScript is the language of the web. It makes websites interactive by responding to user actions, updating content dynamically, and communicating with servers. 


console.log("Hello JavaScript!");

2. Variables (let, const)

Beginner

Variables store data in memory. Use let for changeable values and const for fixed values. 


let name = "Ramesh";
const age = 20;
console.log(name, age);

3. Data Types

Beginner

JavaScript supports strings, numbers, booleans, arrays, objects, null, and undefined. 


let text = "JS";
let num = 10;
let isOk = true;
let arr = [1,2,3];
let obj = { lang: "JS" };
console.log(text, num, isOk, arr, obj);

4. Operators

Beginner

Operators perform operations on values such as arithmetic, comparison, logical, and ternary operations. 


let a = 10, b = 5;
console.log(a + b);
console.log(a > b);
console.log(a === "10");

5. Conditional Statements

Beginner

Conditional statements allow programs to make decisions using if, else if, else, and switch. 


let marks = 80;
if (marks >= 75) {
  console.log("Passed");
} else {
  console.log("Failed");
}

6. Loops

Beginner

Loops repeat a block of code multiple times. Common loops include for and while. 


for (let i = 1; i <= 5; i++) {
  console.log(i);
}

7. Functions

Beginner

Functions are reusable blocks of code that perform specific tasks. 


function add(a, b) {
  return a + b;
}
console.log(add(3, 4));

8. Arrow Functions

Intermediate

Arrow functions provide shorter syntax and are commonly used in callbacks. 


const square = n => n * n;
console.log(square(5));

9. Arrays

Beginner

Arrays store multiple values in a single variable. 


let nums = [10, 20, 30];
nums.push(40);
console.log(nums);

10. Objects

Beginner

Objects store data as key-value pairs and represent real-world entities. 


let user = {
  name: "Ramesh",
  age: 20
};
console.log(user.name, user.age);

11. Arrays – Common Methods

Beginner

Arrays store ordered data. JavaScript provides powerful methods like map, filter, reduce, and forEach to process data efficiently. 


        let nums = [1,2,3,4,5];
        let doubled = nums.map(n => n*2);
        let evens = nums.filter(n => n%2===0);
        console.log(doubled, evens);

12. Objects & Properties

Beginner

Objects represent real-world entities using key–value pairs. Properties can be accessed using dot or bracket notation. 


let student = {
  name: "Ramesh",
  age: 20,
  course: "JS"
};
console.log(student.name);

13. Functions & Scope

Beginner

Scope defines where variables are accessible. JavaScript has global, function, and block scope. 


let x = 10;
function test(){
  let y = 5;
  console.log(x + y);
}
test();

14. Arrow Functions

Intermediate

Arrow functions provide a shorter syntax and do not have their own this. They are widely used in callbacks. 


const greet = name => "Hello " + name;
console.log(greet("JS"));

15. DOM Basics

Intermediate

The DOM represents the webpage as a tree. JavaScript can select and modify elements dynamically. 


const title = document.querySelector("h1");
title.style.color = "orange";

16. Events

Intermediate

Events allow JavaScript to respond to user actions like clicks, typing, and mouse movement. 


document.addEventListener("click", () => {
  console.log("Page clicked");
});

17. Callbacks

Intermediate

Callbacks are functions passed as arguments to other functions. They are commonly used for async operations. 


function process(data, callback){
  callback(data);
}
process("Done", console.log);

18. Promises

Intermediate

Promises handle asynchronous operations and avoid callback hell. States: pending, fulfilled, rejected. 


const p = new Promise(resolve => {
  setTimeout(() => resolve("Success"), 1000);
});
p.then(console.log);

19. Async / Await

Intermediate

Async/await makes asynchronous code easier to read and write, behaving like synchronous code. 


async function demo(){
  return "Async result";
}
demo().then(console.log);

20. Error Handling

Intermediate

Errors are handled using try, catch, and finally blocks. 


try {
  throw new Error("Something broke");
} catch(e) {
  console.log(e.message);
}

21. JSON

Beginner

JSON (JavaScript Object Notation) is used to store and transfer data. It is commonly used when sending data between browser and server. 


let user = { name: "Ramesh", age: 20 };
let json = JSON.stringify(user);
let back = JSON.parse(json);
console.log(back.name);

22. LocalStorage

Beginner

LocalStorage stores data in the browser permanently. Data remains even after refreshing or closing the browser. 


localStorage.setItem("username", "Ramesh");
console.log(localStorage.getItem("username"));

23. SessionStorage

Beginner

SessionStorage stores data only for one browser tab. Data is cleared when the tab is closed. 


sessionStorage.setItem("temp", "123");
console.log(sessionStorage.getItem("temp"));

24. Timers

Beginner

Timers allow delayed or repeated execution. Use setTimeout and setInterval. 


setTimeout(() => console.log("Hello after 1s"), 1000);

25. Regular Expressions

Intermediate

Regular Expressions are used for pattern matching, validation of emails, passwords, and formats. 


let email = "test@example.com";
let pattern = /^[^@]+@[^@]+\.[a-z]+$/;
console.log(pattern.test(email));

26. JavaScript Modules

Intermediate

Modules split code into files using import and export. This improves maintainability and reuse. 


// module.js
export function add(a,b){ return a+b; }

27. Browser APIs

Intermediate

Browsers provide APIs like Geolocation, Clipboard, Canvas, and Notifications to build rich applications. 


navigator.clipboard.writeText("Copied!");

28. Performance Optimization

Advanced

Optimize JavaScript by reducing DOM updates, debouncing events, and minimizing heavy loops. 


function debounce(fn, delay){
  let t;
  return () => {
    clearTimeout(t);
    t = setTimeout(fn, delay);
  };
}

29. Security Basics

Advanced

Prevent XSS by never inserting raw user HTML. Use textContent instead of innerHTML. 


let input = "";
document.body.textContent = input;

30. Best Practices

Advanced

Write clean code: use const, small functions, proper naming, and comments. 


const MAX_USERS = 100;
function getUser(){ return "Admin"; }

Mini Project: Todo App

Project

This mini project helps you practice DOM manipulation, events, arrays, and localStorage by building a simple Todo app. 


<input id="task" placeholder="Enter task">
<button onclick="addTodo()">Add</button>
<ul id="list"></ul>

<script>
function addTodo(){
  const task = document.getElementById("task").value;
  if(!task) return;
  const li = document.createElement("li");
  li.textContent = task;
  document.getElementById("list").appendChild(li);
  document.getElementById("task").value = "";
}
</script>
