Python Full Roadmap

Learn Python step by step with clear explanations, real examples, and practical understanding.

1. Introduction to Python

Beginner

Description:
Python is a high-level, interpreted programming language known for readability and a large ecosystem. It is widely used in web development, data science, automation, scripting, and artificial intelligence, making it one of the most versatile and beginner-friendly languages. 


print("Hello, Python!")  # run with: python file.py

2. Installing Python & Tools

Beginner

Description:
Python can be installed from python.org or using package managers. Developers commonly use VS Code for coding, virtual environments (venv) to isolate dependencies, and pip to install libraries required for projects. 


# create virtual environment
python -m venv venv

# activate (Windows)
venv\Scripts\activate

# install packages
pip install requests numpy pandas

3. Syntax & Variables

Beginner

Description:
Python uses indentation instead of braces to define code blocks, which makes it clean and readable. Variables are dynamically typed, meaning you don’t need to declare their type explicitly, and naming follows snake_case convention. 


name = "Ramesh"
age = 20
is_active = True

print(f"{name} is {age} years old")

4. Data Types

Beginner

Description:
Python provides core data types such as int, float, str, bool, list, tuple, set, dict, and None. These define how data is stored and what operations can be performed, and type() can be used to inspect them. 


x = 10
y = 3.14
s = "text"
lst = [1,2,3]
tpl = (1,2)
st = {1,2}
d = {"a":1}

print(type(x), type(s))

4.1 Input & Output

Beginner

Description:
Input and output allow interaction between the user and program. Python uses input() to take user input and print() to display output, and values are often converted using int() or float(). 


name = input("Enter your name: ")
age = int(input("Enter your age: "))

print(f"Hello {name}, you are {age} years old.")

5. Operators

Beginner

Description:
Operators are used to perform operations on values, including arithmetic, comparison, logical, assignment, and membership operators. They are essential for calculations and decision-making in programs. 


a = 7
b = 3

print(a + b, a // b, a % b)
print(3 in [1,2,3])

6. Control Flow

Beginner

Description:
Control flow statements allow programs to make decisions using if, elif, and else. Python uses indentation instead of braces, making the code more readable and structured while controlling execution based on conditions. 


x = 10

if x > 10:
    print("Greater")
elif x == 10:
    print("Equal")
else:
    print("Smaller")

6.1 Type Conversion

Beginner

Description:
Type conversion is used to convert values from one data type to another using built-in functions like int(), float(), and str(). This is especially useful when handling user input or performing calculations. 


x = "100"

print(int(x) + 50)
print(float("3.14"))
print(str(25) + " apples")

7. Loops

Beginner

Description:
Loops are used to repeat a block of code multiple times. Python provides for loops for iterating over sequences and while loops for condition-based repetition, making programs efficient and reducing code duplication. 


for i in range(5):
    print(i)

i = 0
while i < 5:
    print(i)
    i += 1

7.1 Nested Conditions

Beginner

Description:
Nested conditions allow placing one if statement inside another to evaluate multiple levels of logic. This is useful when decisions depend on more than one condition. 


x = 10

if x > 0:
    if x % 2 == 0:
        print("Positive even")
    else:
        print("Positive odd")

7.2 Loop Control Statements

Beginner

Description:
Loop control statements modify the execution of loops. break exits the loop early, continue skips the current iteration, and else runs when the loop completes normally without interruption. 


for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Completed")

8. Functions

Beginner

Description:
Functions are reusable blocks of code that perform specific tasks. They improve code organization, reduce repetition, and can accept inputs and return outputs, making programs modular and scalable. 


def greet(name="User"):
    return f"Hello, {name}"

print(greet("Ramesh"))

9. Scope & Lifetime

Intermediate

Description:
Python follows LEGB scope rules (Local, Enclosing, Global, Built-in) to determine variable access. Understanding scope helps avoid unexpected behavior and manage variable lifetimes correctly. 


x = "global"

def outer():
    x = "local"
    print(x)

outer()
print(x)

10. Modules & Packages

Intermediate

Description:
Modules and packages help organize code into separate files and folders. They improve code reusability and maintainability, allowing developers to structure large projects efficiently. 


# utils.py
def add(a, b):
    return a + b

# main.py
from utils import add
print(add(2, 3))

11. Virtual Environments & pip

Intermediate

Description:
Virtual environments allow you to isolate project dependencies so that different projects don’t conflict with each other. pip is Python’s package manager used to install, update, and manage external libraries required for development. 


python -m venv venv
venv\Scripts\activate
pip install requests
pip freeze > requirements.txt

12. Data Structures

Intermediate

Description:
Python provides powerful data structures like lists, tuples, sets, and dictionaries. Each serves a different purpose such as ordered storage, immutability, uniqueness, or key-value mapping, making data handling efficient and flexible. 


lst = [1,2,3]
tpl = (1,2,3)
s = {1,2,3}
d = {"a":1, "b":2}

print(lst, tpl, s, d)

13. List Comprehensions

Intermediate

Description:
List comprehensions provide a concise way to create lists using a single line of code. They are faster and more readable than traditional loops and are commonly used in real-world Python code. 


squares = [x*x for x in range(10)]
print(squares)

14. Strings & Formatting

Intermediate

Description:
Strings are immutable sequences used to handle text data. Python provides powerful formatting options like f-strings, along with methods such as split, join, strip, and replace for efficient string manipulation. 


name = "Ramesh"
age = 20

print(f"{name} is {age} years old")

15. File I/O

Intermediate

Description:
File input/output allows reading from and writing to files. Using the with statement ensures files are properly opened and closed, preventing memory leaks and improving safety. 


with open("file.txt", "w") as f:
    f.write("Hello Python")

with open("file.txt", "r") as f:
    print(f.read())

16. Exceptions & Error Handling

Intermediate

Description:
Exception handling allows programs to deal with runtime errors without crashing. Using try, except, and finally blocks ensures errors are handled gracefully and improves program stability. 


try:
    x = int("abc")
except ValueError:
    print("Conversion error")
finally:
    print("Done")

16.1 Debugging & Logging

Intermediate

Description:
Debugging helps identify and fix issues in code, while logging records program activity. Tools like print(), type(), and logging modules help track and understand program behavior during execution. 


x, y = 10, 20
print(x, y, sep=" | ")
print(type(x))

17. Classes & Objects (OOP)

Advanced

Description:
Object-Oriented Programming allows modeling real-world entities using classes and objects. It organizes code into reusable structures with attributes and methods, making applications scalable and maintainable. 


class Person:
    def __init__(self, name):
        self.name = name

p = Person("Ramesh")
print(p.name)

18. Inheritance & Polymorphism

Advanced

Description:
Inheritance allows one class to reuse properties of another, while polymorphism enables the same method to behave differently in different classes. These concepts are essential for building flexible and reusable systems. 


class Animal:
    def speak(self):
        return "sound"

class Dog(Animal):
    def speak(self):
        return "bark"

print(Dog().speak())

19. Decorators

Advanced

Description:
Decorators allow modifying the behavior of functions without changing their code. They are commonly used for logging, timing, authentication, and caching in real-world applications. 


def decorator(func):
    def wrapper():
        print("Before")
        func()
        print("After")
    return wrapper

@decorator
def say():
    print("Hello")

say()

20. Generators & Iterators

Advanced

Description:
Generators produce values one at a time using yield, making them memory efficient. Iterators allow looping over data structures, and together they are useful for handling large datasets and streams. 


def count(n):
    for i in range(n):
        yield i

for i in count(3):
    print(i)

21. Context Managers

Advanced

Description:
Context managers are used to manage resources like files or connections safely. The with statement ensures proper setup and cleanup, even if errors occur, making code more reliable and clean. 


with open("file.txt", "r") as f:
    data = f.read()
    print(data)

22. Lambda & Functional Programming

Advanced

Description:
Lambda functions are small anonymous functions used for short operations. Combined with map, filter, and reduce, they allow functional-style programming for concise and efficient data processing. 


nums = [1,2,3,4]

squared = list(map(lambda x: x*x, nums))
print(squared)

23. Regular Expressions

Advanced

Description:
Regular expressions are used for pattern matching and text validation. They are powerful tools for tasks like email validation, searching patterns, and data cleaning. 


import re

pattern = r"\d+"
print(re.findall(pattern, "Age 20, Year 2024"))

23.1 OS & File Paths

Advanced

Description:
Python provides os and pathlib modules to interact with the operating system, manage directories, and handle file paths in a platform-independent way. 


import os

print(os.getcwd())
print(os.listdir())

24. JSON & CSV Handling

Advanced

Description:
JSON and CSV are common data formats used for storing and transferring data. Python provides built-in modules to read, write, and convert these formats easily. 


import json

data = {"name": "Ramesh"}
json_str = json.dumps(data)

print(json.loads(json_str))

25. Date & Time

Intermediate

Description:
The datetime module is used to work with dates and times. It allows formatting, calculations, and handling time-based operations in applications. 


from datetime import datetime

now = datetime.now()
print(now)

26. Networking (Requests)

Intermediate

Description:
The requests library is used to send HTTP requests to APIs. It is widely used to fetch data from external services and integrate applications with web-based systems. 


import requests

res = requests.get("https://jsonplaceholder.typicode.com/todos/1")
print(res.json())

27. Asyncio & Concurrency

Advanced

Description:
Asyncio allows writing asynchronous code using async and await. It is useful for handling multiple tasks like network calls efficiently without blocking execution. 


import asyncio

async def hello():
    print("Hello")

asyncio.run(hello())

Testing ensures your code works correctly, while debugging helps find and fix errors. Tools like unittest, pytest, and logging are essential for writing reliable applications.

30. NumPy (Intro)

Intermediate

Description:
NumPy provides fast numerical operations and array handling. It is the foundation for scientific computing, data analysis, and machine learning in Python. 


import numpy as np

arr = np.array([1,2,3])
print(arr * 2)

31. Pandas (Intro)

Intermediate

Description:
Pandas is a powerful library used for working with structured data like tables. It provides DataFrames that allow filtering, grouping, cleaning, and analyzing data efficiently, making it essential for data science and analytics. 


import pandas as pd

df = pd.DataFrame({
  "name": ["Ramesh", "Sita"],
  "age": [20, 22]
})

print(df)

31.1 Command Line Arguments

Intermediate

Description:
Command line arguments allow passing inputs directly when running a Python script. This is useful for building CLI tools and automation scripts that accept dynamic input from users. 


import sys

print(sys.argv)

32. Web Development (Flask Intro)

Advanced

Description:
Flask is a lightweight web framework used to build backend applications and APIs. It allows defining routes, handling requests, and returning responses, making it a great starting point for web development with Python. 


from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello Flask"

app.run()
