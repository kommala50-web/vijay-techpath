CSS Full Roadmap

Learn CSS from basics to advanced layout, animation, responsiveness, and modern UI techniques. Click Show Code to view examples.

1. Introduction to CSS

Beginner

CSS (Cascading Style Sheets) is used to style HTML elements. It controls colors, fonts, layouts, spacing, animations, and responsiveness while separating content from presentation. 


body {
  background-color: lightblue;
}
h1 {
  color: navy;
}

2. CSS Syntax & Selectors

Beginner

CSS selectors target HTML elements. Common selectors include element, class, ID, group, and descendant selectors. 


h1 { color: red; }
.intro { font-size: 18px; }
#main { background-color: yellow; }
div p { color: green; }

3. Colors & Backgrounds

Beginner

CSS supports colors via keywords, HEX, RGB, HSL and backgrounds using images, gradients, size, and positioning. 


div {
  background: linear-gradient(to right, red, yellow);
}

4. Fonts & Text

Beginner

Fonts control typography using font-family, size, weight, spacing, alignment, and decoration. 


p {
  font-family: Arial;
  font-size: 16px;
  text-align: center;
}

5. Box Model

Beginner

Every element has content, padding, border, and margin. box-sizing controls width calculation. 


div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
  box-sizing: border-box;
}

6. Display & Visibility

Beginner

display controls layout type, visibility hides elements while preserving space. 


.hidden { display: none; }
.invisible { visibility: hidden; }

7. Positioning

Beginner

Description:
CSS positioning (static, relative, absolute, fixed, sticky) determines how elements are placed in the document. top, left, right, bottom move positioned elements. 


div { position: relative; top: 10px; left: 20px; }
header { position: fixed; top: 0; width: 100%; }

8. Floating & Clear

Beginner

Description:
float positions elements left or right. clear prevents elements from wrapping around floated elements. Used in old layout techniques. 


img { float: left; margin: 10px; }
div::after { content: ""; clear: both; display: table; }

9. Flexbox

Intermediate

Description:
Flexbox is a modern layout system for 1D layouts. Flex containers align items horizontally or vertically using flex-direction, justify-content, align-items, flex-wrap, and more. 


.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}
.item { flex: 1 1 200px; }

10. Grid

Intermediate

Description:
CSS Grid is for 2D layouts with rows and columns. It offers precise control over placement using grid-template-rows, grid-template-columns, grid-gap, and grid-area. 


.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}
.item { background-color: lightblue; }

11. Pseudo-classes & Pseudo-elements

Intermediate

Description:
Pseudo-classes (:hover, :focus, :nth-child) select states. Pseudo-elements (::before, ::after, ::first-letter) insert content or style parts of elements. 


a:hover { color: red; }
li:nth-child(odd) { background: #eee; }
p::first-letter { font-size: 24px; color: blue; }

12. Transitions

Intermediate

Description:
CSS transitions create smooth animations between property changes, specifying duration, timing-function, and delay. 


div {
  background-color: blue;
  transition: background-color 0.5s ease;
}
div:hover { background-color: green; }

13. Animations

Intermediate

Description:
CSS animations use keyframes to create complex motion. Define @keyframes and use animation property for duration, iteration, and timing. 


@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
div { animation: slide 2s infinite alternate; }

14. Transforms

Intermediate

Description:
Transform rotates, scales, skews, and translates elements in 2D or 3D space. 


div { transform: rotate(45deg); scale: 1.2; }

15. Filters & Effects

Intermediate

Description:
CSS filters adjust images or elements (blur, grayscale, brightness). Shadows and opacity create visual effects. 


img { filter: grayscale(50%); }
div { box-shadow: 2px 2px 10px rgba(0,0,0,0.5); }
p { opacity: 0.8; }

16. Units & Measurements

Beginner

Description:
CSS supports px, em, rem, %, vh, vw, fr, and more for flexible and responsive sizing. 


div { width: 50%; font-size: 1.2em; margin: 10px 5vw; }

17. Media Queries (Responsive)

Intermediate

Description:
Media queries adapt layout for different screen sizes, devices, and orientations using @media rules. 


@media (max-width: 768px) {
  .container { flex-direction: column; }
  body { font-size: 14px; }
}

18. CSS Variables

Intermediate

Description:
CSS variables (custom properties) allow reusability and easier maintenance. Defined with --name and accessed with var(--name). 


:root {
  --main-color: #00b4d8;
}
p { color: var(--main-color); }

19. Gradients

Intermediate

Description:
CSS gradients create smooth color transitions. Linear and radial gradients are commonly used for backgrounds and overlays. 


div {
  background: linear-gradient(to right, red, yellow);
}
section {
  background: radial-gradient(circle, blue, lightblue);
}

20. Shorthand Properties

Beginner

Description:
Shorthand combines multiple properties into one, like margin, padding, border, font, background, etc. 


div {
  margin: 10px 20px 30px 40px;
  padding: 5px 10px;
  border: 2px solid black;
}

21. CSS Functions

Intermediate

Description:
CSS provides functions like calc(), min(), max(), clamp(), and url() for dynamic values. 


div { width: calc(100% - 40px); }
p { font-size: clamp(14px, 2vw, 20px); }

22. Bonus: Accessibility & Best Practices

Advanced

Description:
Use semantic HTML, proper color contrast, readable fonts, responsive layouts, ARIA attributes, and meaningful focus states for accessibility and SEO. 


button { outline: 2px solid #00b4d8; }
img { alt: "Description of image"; }
a { focus { outline: 2px dashed blue; } }
