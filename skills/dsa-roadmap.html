DSA Full Roadmap (Java)

Learn Data Structures and Algorithms step-by-step using Java. Each topic is explained in simple words with clean Java code examples. Click Show Code to see the implementation.

1. Introduction to DSA & Complexity

Beginner

Data Structures and Algorithms (DSA) help us write programs that run faster and use less memory. When a problem has many possible solutions, DSA helps us choose the most efficient one.

Time complexity tells us how the running time of a program grows as input size increases. Space complexity tells us how much extra memory the program uses. Big-O notation is used to express both. 


class Main {
    public static void main(String[] args) {
        int n = 5;

        // O(n^2) time complexity
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print(i + j + " ");
            }
            System.out.println();
        }
    }
}

2. Arrays

Beginner

An array stores multiple values of the same data type in a single variable. All elements are stored in continuous memory locations.

In Java, array indexing starts from 0. Accessing an element using its index is very fast (O(1)), but inserting or deleting elements in the middle is slow because elements must be shifted. 


class Main {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5};

        // Update element
        arr[2] = 10;

        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i] + " ");
        }
    }
}

4. Singly Linked List

Beginner

A singly linked list is a collection of nodes where each node stores data and a reference to the next node. Unlike arrays, linked lists do not store elements in continuous memory, so insertion and deletion are easier. However, accessing elements requires traversal from the head. 


class Node {
    int data;
    Node next;

    Node(int d) {
        data = d;
        next = null;
    }
}

class Main {
    public static void main(String[] args) {
        Node head = new Node(10);
        head.next = new Node(20);

        Node temp = head;
        while (temp != null) {
            System.out.print(temp.data + " ");
            temp = temp.next;
        }
    }
}

5. Doubly Linked List

Beginner

A doubly linked list stores data along with two references: one to the previous node and one to the next node. This allows traversal in both directions and makes deletion easier, but it uses more memory than a singly linked list. 


class Node {
    int data;
    Node prev, next;

    Node(int d) {
        data = d;
        prev = next = null;
    }
}

class Main {
    public static void main(String[] args) {
        Node head = new Node(10);
        Node second = new Node(20);

        head.next = second;
        second.prev = head;

        Node temp = head;
        while (temp != null) {
            System.out.print(temp.data + " ");
            temp = temp.next;
        }
    }
}

6. Stack

Beginner

A stack follows the LIFO principle (Last In, First Out). Elements are added using push and removed using pop. Stacks are commonly used in recursion, undo operations, expression evaluation, and backtracking problems. 


import java.util.Stack;

class Main {
    public static void main(String[] args) {
        Stack stack = new Stack<>();

        stack.push(10);
        stack.push(20);

        System.out.println(stack.peek()); // 20
        stack.pop();
        System.out.println(stack.peek()); // 10
    }
}

7. Queue

Beginner

A queue follows the FIFO principle (First In, First Out). Elements are inserted at the rear and removed from the front. Queues are used in scheduling, breadth-first search, and real-world systems like task processing. 


import java.util.LinkedList;
import java.util.Queue;

class Main {
    public static void main(String[] args) {
        Queue q = new LinkedList<>();

        q.add(10);
        q.add(20);

        System.out.println(q.peek()); // 10
        q.remove();
        System.out.println(q.peek()); // 20
    }
}

8. Recursion

Beginner

Recursion is a technique where a function calls itself to solve a problem. Every recursive function must have a base case to stop execution. Recursion is widely used in tree traversal, divide-and-conquer, and dynamic programming problems. 


class Main {
    static int factorial(int n) {
        if (n == 0) return 1;
        return n * factorial(n - 1);
    }

    public static void main(String[] args) {
        System.out.println(factorial(5)); // 120
    }
}

9. Searching (Linear & Binary)

Beginner

Searching is the process of finding an element in a collection. Linear search checks elements one by one, while binary search works only on sorted data and reduces search time by half each step. 


import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        int[] arr = {1, 3, 5, 7, 9};

        // Linear search
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] == 5)
                System.out.println("Found at index " + i);
        }

        // Binary search
        int idx = Arrays.binarySearch(arr, 7);
        System.out.println("Binary search index: " + idx);
    }
}

10. Sorting (Bubble Sort)

Beginner

Sorting arranges data in ascending or descending order. Bubble sort is a simple sorting algorithm that repeatedly swaps adjacent elements if they are in the wrong order. Although easy to understand, it is inefficient for large data. 


class Main {

    }
}

22. Binary Search

Beginner

Binary Search is an efficient searching technique that works on sorted arrays. It repeatedly divides the search space into half, reducing time complexity from O(n) to O(log n). 


class Main {
    public static void main(String[] args) {
        int[] arr = {1,3,5,7,9};
        int target = 7;

        int l = 0, r = arr.length - 1;
        while (l <= r) {
            int mid = (l + r) / 2;
            if (arr[mid] == target) {
                System.out.println(mid);
                return;
            } else if (arr[mid] < target) l = mid + 1;
            else r = mid - 1;
        }
        System.out.println("Not found");
    }
}

24. Merge Sort

Intermediate

Merge Sort is a divide-and-conquer algorithm that splits the array into halves, sorts each half, and then merges them. It guarantees O(n log n) time complexity and is very efficient for large data. 


import java.util.Arrays;

class Main {
    static void mergeSort(int[] arr) {
        if (arr.length < 2) return;

        int mid = arr.length / 2;
        int[] left = Arrays.copyOfRange(arr, 0, mid);
        int[] right = Arrays.copyOfRange(arr, mid, arr.length);

        mergeSort(left);
        mergeSort(right);

        int i = 0, j = 0, k = 0;
        while (i < left.length && j < right.length)
            arr[k++] = left[i] < right[j] ? left[i++] : right[j++];
        while (i < left.length) arr[k++] = left[i++];
        while (j < right.length) arr[k++] = right[j++];
    }

    public static void main(String[] args) {
        int[] arr = {5,2,9,1};
        mergeSort(arr);
        System.out.println(Arrays.toString(arr));
    }
}

25. Quick Sort

Intermediate

Quick Sort selects a pivot element and partitions the array so that smaller elements go to the left and larger ones to the right. It is very fast in practice and widely used. 


class Main {
    static int partition(int[] arr, int l, int r) {
        int pivot = arr[r];
        int i = l;
        for (int j = l; j < r; j++) {
            if (arr[j] < pivot) {
                int t = arr[i];
                arr[i++] = arr[j];
                arr[j] = t;
            }
        }
        int t = arr[i];
        arr[i] = arr[r];
        arr[r] = t;
        return i;
    }

    static void quickSort(int[] arr, int l, int r) {
        if (l < r) {
            int p = partition(arr, l, r);
            quickSort(arr, l, p - 1);
            quickSort(arr, p + 1, r);
        }
    }

    public static void main(String[] args) {
        int[] arr = {6,3,8,5,2};
        quickSort(arr, 0, arr.length - 1);
        for (int x : arr) System.out.print(x + " ");
    }
}

26. Greedy Algorithms

Intermediate

Greedy algorithms make the best possible choice at each step, hoping it leads to an optimal solution. They are simple and fast but work only for problems that satisfy the greedy property. 


import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        int[] coins = {1, 2, 5};
        int amount = 11;
        int count = 0;

        for (int i = coins.length - 1; i >= 0; i--) {
            count += amount / coins[i];
            amount %= coins[i];
        }
        System.out.println(count);
    }
}

27. Dynamic Programming (Fibonacci)

Intermediate

Dynamic Programming avoids repeated calculations by storing results of subproblems. Fibonacci is a classic example where DP improves performance from exponential to linear time. 


class Main {
    public static void main(String[] args) {
        int n = 10;
        int[] dp = new int[n + 1];
        dp[0] = 0; dp[1] = 1;

        for (int i = 2; i <= n; i++)
            dp[i] = dp[i - 1] + dp[i - 2];

        System.out.println(dp[n]);
    }
}

28. 0/1 Knapsack

Advanced

The 0/1 Knapsack problem finds the maximum value that can be placed in a bag with limited capacity. Each item can be chosen only once. Dynamic Programming is used to solve it efficiently. 


class Main {
    public static void main(String[] args) {
        int[] wt = {1,3,4,5};
        int[] val = {1,4,5,7};
        int W = 7;

        int[][] dp = new int[wt.length + 1][W + 1];

        for (int i = 1; i <= wt.length; i++) {
            for (int w = 1; w <= W; w++) {
                if (wt[i-1] <= w)
                    dp[i][w] = Math.max(
                        val[i-1] + dp[i-1][w - wt[i-1]],
                        dp[i-1][w]
                    );
                else
                    dp[i][w] = dp[i-1][w];
            }
        }
        System.out.println(dp[wt.length][W]);
    }
}

30. Dijkstra’s Algorithm

Advanced

Dijkstra’s algorithm finds the shortest path from a source node to all other nodes in a weighted graph with non-negative edges. It uses a priority queue to always expand the nearest node first. 


import java.util.*;

class Main {
    public static void main(String[] args) {
        int V = 3;
        List[] graph = new ArrayList[V];
        for (int i = 0; i < V; i++) graph[i] = new ArrayList<>();

        graph[0].add(new int[]{1,4});
        graph[0].add(new int[]{2,2});
        graph[2].add(new int[]{1,1});

        int[] dist = new int[V];
        Arrays.fill(dist, Integer.MAX_VALUE);
        dist[0] = 0;

        PriorityQueue pq =
            new PriorityQueue<>(Comparator.comparingInt(a -> a[0]));
        pq.add(new int[]{0,0});

        while (!pq.isEmpty()) {
            int[] cur = pq.poll();
            int d = cur[0], u = cur[1];
            if (d > dist[u]) continue;

            for (int[] e : graph[u]) {
                int v = e[0], w = e[1];
                if (dist[u] + w < dist[v]) {
                    dist[v] = dist[u] + w;
                    pq.add(new int[]{dist[v], v});
                }
            }
        }

        System.out.println(Arrays.toString(dist));
    }
}

31. Binary Search Tree (BST)

Intermediate

A Binary Search Tree is a tree where the left child contains smaller values and the right child contains larger values. This property allows fast searching, insertion, and deletion with average time complexity O(log n). 


class Node {
    int data;
    Node left, right;
    Node(int d) { data = d; }
}

class Main {
    static Node insert(Node root, int val) {
        if (root == null) return new Node(val);
        if (val < root.data) root.left = insert(root.left, val);
        else root.right = insert(root.right, val);
        return root;
    }

    static void inorder(Node root) {
        if (root == null) return;
        inorder(root.left);
        System.out.print(root.data + " ");
        inorder(root.right);
    }

    public static void main(String[] args) {
        Node root = null;
        root = insert(root, 10);
        root = insert(root, 5);
        root = insert(root, 15);
        inorder(root);
    }
}

35. Breadth First Search (BFS)

Intermediate

BFS explores a graph level by level using a queue. It is commonly used to find the shortest path in unweighted graphs and to check connectivity. 


import java.util.*;

class Main {
    public static void main(String[] args) {
        List[] g = new ArrayList[3];
        for (int i = 0; i < 3; i++) g[i] = new ArrayList<>();
        g[0].add(1); g[1].add(2);

        Queue q = new LinkedList<>();
        boolean[] vis = new boolean[3];
        q.add(0); vis[0] = true;

        while (!q.isEmpty()) {
            int v = q.poll();
            System.out.print(v + " ");
            for (int n : g[v])
                if (!vis[n]) { vis[n] = true; q.add(n); }
        }
    }
}

36. Depth First Search (DFS)

Intermediate

DFS explores a graph deeply before backtracking. It is implemented using recursion or a stack and is useful for cycle detection and path finding. 


import java.util.*;

class Main {
    static void dfs(int v, boolean[] vis, List[] g) {
        vis[v] = true;
        System.out.print(v + " ");
        for (int n : g[v])
            if (!vis[n]) dfs(n, vis, g);
    }

    public static void main(String[] args) {
        List[] g = new ArrayList[3];
        for (int i = 0; i < 3; i++) g[i] = new ArrayList<>();
        g[0].add(1); g[1].add(2);
        dfs(0, new boolean[3], g);
    }
}

37. Sliding Window Technique

Intermediate

Sliding Window is used to efficiently process subarrays or substrings by maintaining a moving window. It reduces time complexity from O(n²) to O(n) for many problems. 


class Main {
    public static void main(String[] args) {
        int[] arr = {1,2,3,4,5};
        int k = 3, sum = 0;
        for (int i = 0; i < k; i++) sum += arr[i];
        int max = sum;

        for (int i = k; i < arr.length; i++) {
            sum += arr[i] - arr[i-k];
            max = Math.max(max, sum);
        }
        System.out.println(max);
    }
}

38. Two Pointers Technique

Intermediate

The two pointers technique uses two indices to process arrays efficiently. It is commonly used for pair-sum problems, removing duplicates, and searching in sorted arrays. 


class Main {
    public static void main(String[] args) {
        int[] arr = {1,2,3,4,5};
        int l = 0, r = arr.length - 1, target = 6;

        while (l < r) {
            int sum = arr[l] + arr[r];
            if (sum == target) {
                System.out.println(l + " " + r);
                break;
            } else if (sum < target) l++;
            else r--;
        }
    }
}

40. Bellman-Ford Algorithm

Advanced

Bellman-Ford finds shortest paths even when negative edge weights exist. It relaxes all edges multiple times and can also detect negative cycles. 


class Main {
    public static void main(String[] args) {
        int V = 3;
        int[] dist = {0, 999, 999};
        int[][] edges = {{0,1,4},{0,2,2},{2,1,-2}};

        for (int i = 0; i < V-1; i++)
            for (int[] e : edges)
                if (dist[e[0]] + e[2] < dist[e[1]])
                    dist[e[1]] = dist[e[0]] + e[2];

        for (int d : dist) System.out.print(d + " ");
    }
}
