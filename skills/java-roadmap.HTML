Java Full Roadmap

Learn Java from scratch to advanced concepts like OOP, multithreading, collections, and real-world applications. Click Show Code to view examples.

1. Introduction to Java

Beginner

Java is a high-level, object-oriented programming language developed by Sun Microsystems. It is platform-independent because Java programs run on the JVM (Java Virtual Machine). Java is widely used in web applications, Android apps, enterprise systems, and backend services. 


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

2. Java Setup & Execution

Beginner

To run Java programs, you need to install the JDK (Java Development Kit). Java source code is compiled using javac and executed using the java command. IDEs like IntelliJ, Eclipse, or VS Code simplify development. 


// Compile
javac HelloWorld.java

// Run
java HelloWorld

3. Data Types & Variables

Beginner

Java has primitive data types such as int, double, char, and boolean. Variables store data in memory and must be declared with a type before use. 


int age = 25;
double salary = 45000.5;
char grade = 'A';
boolean active = true;

System.out.println(age);
System.out.println(salary);
System.out.println(grade);
System.out.println(active);

4. Operators

Beginner

Operators perform operations on variables and values. Java supports arithmetic, relational, logical, bitwise, assignment, and ternary operators for building expressions and conditions. 


int a = 10, b = 5;

System.out.println(a + b);
System.out.println(a > b);
System.out.println(a & b);
System.out.println((a > b) ? a : b);

5. Control Statements

Beginner

Control statements decide the flow of execution. Java provides if, if-else, switch for decisions and for, while, do-while loops for repetition. 


for(int i = 1; i <= 5; i++) {
    if(i == 3) continue;
    System.out.println(i);
}

6. Methods (Functions)

Beginner

Methods are reusable blocks of code that perform a specific task. They help reduce duplication and make programs easier to read and maintain. Methods can return values and accept parameters. 


public static int sum(int a, int b) {
    return a + b;
}

public static void main(String[] args) {
    System.out.println(sum(5, 3));
}

7. Arrays

Beginner

Arrays are used to store multiple values of the same data type in a single variable. Java arrays have a fixed size and allow fast access using index positions. They are commonly used for storing lists, tables, and numerical data. 


int[] numbers = {1, 2, 3, 4, 5};

for (int n : numbers) {
    System.out.println(n);
}

8. Strings

Beginner

Strings are used to store text data in Java. Java provides the String class with built-in methods like length(), substring(), equals(), and concat(). Strings are immutable, meaning their values cannot be changed after creation. 


String msg = "Hello";
String full = msg.concat(" Java");

System.out.println(full);
System.out.println(full.length());

9. Classes & Objects

Beginner

Java is an object-oriented language. A class is a blueprint that defines properties and behaviors. An object is an instance of a class created using the new keyword. Classes help organize code and model real-world entities. 


class Student {
    String name;
    int age;

    void display() {
        System.out.println(name + " " + age);
    }
}

public class Main {
    public static void main(String[] args) {
        Student s = new Student();
        s.name = "Alice";
        s.age = 20;
        s.display();
    }
}

10. Inheritance

Intermediate

Inheritance allows one class to reuse the properties and methods of another class. It is implemented using the extends keyword. Inheritance promotes code reusability and establishes a parent-child relationship. 


class Animal {
    void eat() {
        System.out.println("Eating...");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Barking...");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog d = new Dog();
        d.eat();
        d.bark();
    }
}

11. Polymorphism

Intermediate

Polymorphism allows a method to behave differently based on the object calling it. Java supports method overloading (compile-time polymorphism) and method overriding (runtime polymorphism). 


class Shape {
    void draw() {
        System.out.println("Drawing shape");
    }
}

class Circle extends Shape {
    @Override
    void draw() {
        System.out.println("Drawing circle");
    }
}

public class Main {
    public static void main(String[] args) {
        Shape s = new Circle();
        s.draw();
    }
}

12. Encapsulation

Intermediate

Encapsulation protects data by restricting direct access to class variables. It is achieved using private variables and public getter and setter methods. This improves security and control over data. 


class Account {
    private double balance;

    public void setBalance(double b) {
        balance = b;
    }

    public double getBalance() {
        return balance;
    }
}

public class Main {
    public static void main(String[] args) {
        Account acc = new Account();
        acc.setBalance(5000);
        System.out.println(acc.getBalance());
    }
}

13. Interfaces

Intermediate

An interface defines a contract that a class must follow. It contains abstract methods only (by default). A class can implement multiple interfaces, which helps achieve multiple inheritance in Java. 


interface Vehicle {
    void move();
}

class Car implements Vehicle {
    public void move() {
        System.out.println("Car is moving");
    }
}

public class Main {
    public static void main(String[] args) {
        Vehicle v = new Car();
        v.move();
    }
}

14. Abstract Classes

Intermediate

Abstract classes are partially implemented classes. They can have both abstract methods and normal methods. Objects of abstract classes cannot be created directly. 


abstract class Shape {
    abstract void draw();
}

class Square extends Shape {
    void draw() {
        System.out.println("Drawing Square");
    }
}

public class Main {
    public static void main(String[] args) {
        Shape s = new Square();
        s.draw();
    }
}

15. Packages

Beginner

Packages are used to group related classes and interfaces together. They help organize large projects, avoid name conflicts, and improve code readability. Java provides built-in packages like java.util and java.io. 


package mypackage;

public class Demo {
    public void show() {
        System.out.println("Inside package");
    }
}

16. Access Modifiers

Beginner

Access modifiers control visibility of classes, methods, and variables. Java provides four types: public, private, protected, and default. They help protect data and enforce encapsulation. 


class Test {
    private int x = 10;

    public int getX() {
        return x;
    }
}

public class Main {
    public static void main(String[] args) {
        Test t = new Test();
        System.out.println(t.getX());
    }
}

17. Constructors

Beginner

Constructors are special methods used to initialize objects. They have the same name as the class and no return type. Constructors are called automatically when an object is created. 


class Student {
    String name;

    Student(String n) {
        name = n;
    }
}

public class Main {
    public static void main(String[] args) {
        Student s = new Student("Rahul");
        System.out.println(s.name);
    }
}

18. this Keyword

Beginner

The this keyword refers to the current object. It is used to differentiate instance variables from parameters and to call another constructor of the same class. 


class Person {
    String name;

    Person(String name) {
        this.name = name;
    }
}

public class Main {
    public static void main(String[] args) {
        Person p = new Person("Amit");
        System.out.println(p.name);
    }
}

19. static Keyword

Intermediate

Static variables and methods belong to the class, not to objects. They are shared among all instances. Static members can be accessed without creating an object. 


class Counter {
    static int count = 0;

    Counter() {
        count++;
    }
}

public class Main {
    public static void main(String[] args) {
        new Counter();
        new Counter();
        System.out.println(Counter.count);
    }
}

20. final Keyword

Intermediate

The final keyword is used to restrict modification. Final variables become constants, final methods cannot be overridden, and final classes cannot be inherited. 


final class Constants {
    static final double PI = 3.14;
}

public class Main {
    public static void main(String[] args) {
        System.out.println(Constants.PI);
    }
}

21. Exception Handling

Intermediate

Exception handling allows a program to handle runtime errors gracefully instead of crashing. Java uses try, catch, finally, throw, and throws keywords to manage errors like division by zero or invalid input. 


public class Main {
    public static void main(String[] args) {
        try {
            int x = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Cannot divide by zero");
        } finally {
            System.out.println("Program finished");
        }
    }
}

22. Custom Exceptions

Intermediate

Custom exceptions are user-defined exceptions. They are created by extending the Exception class and are useful for application-specific error handling. 


class AgeException extends Exception {
    AgeException(String msg) {
        super(msg);
    }
}

public class Main {
    static void checkAge(int age) throws AgeException {
        if (age < 18)
            throw new AgeException("Not eligible to vote");
    }

    public static void main(String[] args) {
        try {
            checkAge(16);
        } catch (AgeException e) {
            System.out.println(e.getMessage());
        }
    }
}

23. Generics

Intermediate

Generics allow writing reusable and type-safe code. They prevent runtime type errors by checking types at compile time. Commonly used in collections like ArrayList and HashMap. 


class Box {
    T value;
    void set(T value) { this.value = value; }
    T get() { return value; }
}

public class Main {
    public static void main(String[] args) {
        Box box = new Box<>();
        box.set("Hello");
        System.out.println(box.get());
    }
}

24. Collections Framework

Intermediate

The Collections Framework provides ready-made data structures like List, Set, and Map. It helps store, retrieve, and manipulate data efficiently using classes like ArrayList and HashSet. 


import java.util.*;

public class Main {
    public static void main(String[] args) {
        List list = new ArrayList<>();
        list.add(10);
        list.add(20);
        list.add(30);

        for (int x : list)
            System.out.println(x);
    }
}

25. Map Interface

Intermediate

Map stores data in key-value pairs. Keys are unique, while values can be duplicated. HashMap and TreeMap are commonly used implementations. 


import java.util.*;

public class Main {
    public static void main(String[] args) {
        Map map = new HashMap<>();
        map.put("Math", 90);
        map.put("Science", 85);

        System.out.println(map.get("Math"));
    }
}

26. Multithreading

Advanced

Multithreading allows multiple tasks to run concurrently. Java supports threads using the Thread class or Runnable interface. It improves performance and responsiveness. 


class MyThread extends Thread {
    public void run() {
        System.out.println("Thread running");
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread t = new MyThread();
        t.start();
    }
}

27. Synchronization

Advanced

Synchronization prevents multiple threads from accessing shared resources at the same time. It avoids data inconsistency and race conditions. 


class Counter {
    int count = 0;

    synchronized void increment() {
        count++;
    }
}

28. File Handling

Intermediate

File handling allows reading and writing data to files. Java provides File, FileReader, FileWriter, and BufferedReader classes for handling files. 


import java.io.*;

public class Main {
    public static void main(String[] args) throws Exception {
        FileWriter fw = new FileWriter("test.txt");
        fw.write("Hello File");

31. Java 8 Features

Advanced

Java 8 introduced major improvements like lambda expressions, Stream API, Optional class, default methods in interfaces, and a new Date & Time API. These features make code shorter, cleaner, and more readable. 


interface MathOp {
    int apply(int a, int b);
}

public class Main {
    public static void main(String[] args) {
        MathOp add = (a, b) -> a + b;
        System.out.println(add.apply(5, 3));
    }
}

32. Lambda Expressions

Advanced

Lambda expressions provide a short way to write functions. They are mainly used with functional interfaces and are widely used in collections and streams. 


Runnable task = () -> System.out.println("Running thread");
new Thread(task).start();

33. Stream API

Advanced

Stream API allows functional-style operations on collections such as filtering, mapping, and reducing data. It improves readability and performance. 


import java.util.*;

public class Main {
    public static void main(String[] args) {
        List nums = Arrays.asList(1,2,3,4,5);
        nums.stream()
            .filter(n -> n % 2 == 0)
            .forEach(System.out::println);
    }
}

34. Optional Class

Advanced

Optional is used to avoid NullPointerException. It represents a value that may or may not be present and forces developers to handle null cases safely. 


import java.util.Optional;

public class Main {
    public static void main(String[] args) {
        Optional name = Optional.of("Java");
        System.out.println(name.orElse("No Value"));
    }
}

35. Date & Time API

Advanced

Java 8 introduced a new Date & Time API under java.time package. It fixes problems of old Date class and provides immutable, thread-safe date-time objects. 


import java.time.*;

public class Main {
    public static void main(String[] args) {
        LocalDate today = LocalDate.now();
        LocalTime time = LocalTime.now();
        System.out.println(today + " " + time);
    }
}

37. Memory Management

Advanced

Java memory is divided into Stack and Heap. Stack stores method calls and local variables, while Heap stores objects and is managed by GC. 


public class Main {
    public static void main(String[] args) {
        String s = new String("Hello");
        s = null; // eligible for garbage collection
    }
}

38. Garbage Collection

Advanced

Garbage Collector automatically removes unused objects from heap memory. It helps prevent memory leaks and improves application stability. 


public class Main {
    public static void main(String[] args) {
        System.gc(); // Suggest GC
        System.out.println("GC requested");
    }
}

39. Design Patterns

Advanced

Design patterns are reusable solutions to common software design problems. Examples include Singleton, Factory, Observer, and Strategy patterns. 


class Singleton {
    private static Singleton instance;
    private Singleton() {}
    public static Singleton getInstance() {
        if(instance == null)
            instance = new Singleton();
        return instance;
    }
}

42. JDBC (Database Connectivity)

Advanced

JDBC allows Java applications to connect to databases. It uses DriverManager, Connection, Statement, and ResultSet to perform SQL operations. 


import java.sql.*;

public class Main {
    public static void main(String[] args) throws Exception {
        Connection con = DriverManager.getConnection(
            "jdbc:mysql://localhost:3306/testdb",
            "root",
            "password"
        );
        Statement st = con.createStatement();
        ResultSet rs = st.executeQuery("SELECT * FROM students");

        while(rs.next()) {
            System.out.println(rs.getInt(1) + " " + rs.getString(2));
        }
        con.close();
    }
}

43. File Handling

Advanced

Java provides File, FileReader, FileWriter, BufferedReader, and BufferedWriter classes to read and write files efficiently. 


import java.io.*;

public class Main {
    public static void main(String[] args) throws Exception {
        FileWriter fw = new FileWriter("data.txt");
        fw.write("Hello File");
        fw.close();

        BufferedReader br = new BufferedReader(new FileReader("data.txt"));
        System.out.println(br.readLine());
        br.close();
    }
}

44. Reflection API

Advanced

Reflection allows inspecting classes, methods, and fields at runtime. It is commonly used in frameworks like Spring and Hibernate. 


import java.lang.reflect.*;

class Student {
    private String name = "TechPath";
}

public class Main {
    public static void main(String[] args) throws Exception {
        Student s = new Student();
        Field f = s.getClass().getDeclaredField("name");
        f.setAccessible(true);
        System.out.println(f.get(s));
    }
}
