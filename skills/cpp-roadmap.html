C++ Full Roadmap

Learn C++ from fundamentals to advanced concepts like OOP, STL, memory management, and modern C++. Click Show Code to view examples in a popup.

1. C++ Basics & Setup

Beginner

Description:
C++ is a high-performance, general-purpose language that builds on C. It is used in systems programming, game development, embedded systems, and high-performance applications. A C++ program consists of headers, the main() function, and statements within curly braces. Input/output is performed using cin and cout from the <iostream> library. Comments can be single-line (//) or multi-line (/* */).

Example Question: Write a program to input your name and age, then display them using cout.


#include 
using namespace std;

int main() {
    string name;
    int age;
    cout << "Enter your name: ";
    cin >> name;
    cout << "Enter your age: ";
    cin >> age;
    cout << "Hello, " << name << "! You are " << age << " years old." << endl;
    return 0;
}

2. Data Types & Variables

Beginner

Description:
Data types specify what kind of data a variable can hold. Common built-in types include int, char, float, double, and bool. Variables must be declared before use. Constants declared with const or constexpr cannot be changed after initialization. Type modifiers control size and range.

Example Question: Write a program demonstrating constants and type casting.


#include 
using namespace std;

int main() {
    const double PI = 3.14159;
    int radius = 5;
    double area = PI * radius * radius;
    cout << "Area of circle: " << area << endl;

    double num = 7.8;
    int converted = static_cast(num);
    cout << "Converted value: " << converted << endl;
    return 0;
}

3. Operators

Intermediate

Operators are symbols that perform operations on variables and values. C++ supports arithmetic, relational, logical, bitwise, assignment, and conditional operators.

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Bitwise: &, |, ^, ~, <<, >>
  • Ternary: ?:

Example Question: Write a program to demonstrate arithmetic and logical operators.


#include 
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "a + b = " << a + b << endl;
    cout << "(a > b && b > 0) = " << ((a > b) && (b > 0)) << endl;
    return 0;
}

4. Control Flow

Beginner

Control flow statements determine the order of execution. Includes if-else, switch, loops, break, and continue.

Example Question: Print all even numbers from 1 to 20.


#include 
using namespace std;

int main() {
    for(int i = 1; i <= 20; i++) {
        if(i % 2 == 0)
            cout << i << " ";
    }
    return 0;
}

5. Functions

Beginner

Functions are reusable blocks of code that perform a task. They improve readability and reduce duplication.

Example Question: Write a function to calculate factorial using recursion.


#include 
using namespace std;

int factorial(int n) {
    if(n <= 1) return 1;
    return n * factorial(n - 1);
}

int main() {
    cout << factorial(5);
    return 0;
}

6. Arrays & Strings

Beginner

Arrays store multiple values of the same type in contiguous memory. Strings can be handled using the string class.

Example Question: Input numbers into an array and print them.


#include 
using namespace std;

int main() {
    int arr[3] = {1,2,3};
    for(int i : arr)
        cout << i << " ";
    return 0;
}

7. Pointers

Intermediate

Description:
Pointers are variables that store the memory address of another variable. They allow direct memory manipulation and efficient data handling. Key concepts include pointer declaration, initialization, pointer arithmetic, pointers to arrays, and pointers to pointers. The nullptr keyword represents a null pointer. Dynamic memory allocation is done using new and delete.

Example Question: Write a program to swap two numbers using pointers.


#include 
using namespace std;

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(&x, &y);
    cout << x << " " << y;
    return 0;
}

8. References

Intermediate

Description:
References are aliases for existing variables. They must be initialized when declared and cannot be null. References simplify pass-by-reference without explicit pointers.

Example Question: Increment a number using reference parameters.


#include 
using namespace std;

void increment(int &x) {
    x++;
}

int main() {
    int a = 5;
    increment(a);
    cout << a;
    return 0;
}

9. Dynamic Memory & Storage Classes

Intermediate

Description:
Dynamic memory allocation allows memory to be allocated at runtime using new and released using delete. Storage classes define scope, lifetime, and visibility of variables such as auto, static, extern, and register.

Example Question: Create a dynamic array and calculate its sum.


#include 
using namespace std;

int main() {
    int n, sum = 0;
    cin >> n;
    int *arr = new int[n];
    for(int i=0;i> arr[i];
        sum += arr[i];
    }
    cout << sum;
    delete[] arr;
    return 0;
}

10. Structures & Enumerations

Intermediate

Description:
Structures group different data types under one name. Enumerations define a set of named constants. enum class provides type safety.

Example Question: Store and display student information using struct.


#include 
using namespace std;

struct Student {
    string name;
    int age;
};

int main() {
    Student s = {"Alice", 20};
    cout << s.name << " " << s.age;
    return 0;
}

11. Object-Oriented Programming (OOP)

Advanced

Description:
OOP models real-world entities using classes and objects. Core principles include encapsulation, inheritance, polymorphism, and abstraction.

Example Question: Create a base class and override a function in a derived class.


#include 
using namespace std;

class Base {
public:
    virtual void show() {
        cout << "Base";
    }
};

class Derived : public Base {
public:
    void show() override {
        cout << "Derived";
    }
};

int main() {
    Base* b = new Derived();
    b->show();
    delete b;
    return 0;
}

12. Operator Overloading

Advanced

Description:
Operator overloading allows redefining operators for user-defined types. It improves readability and intuitive object interaction.

Example Question: Overload the + operator for a class.


#include 
using namespace std;

class Num {
public:
    int x;
    Num(int a) : x(a) {}
    Num operator+(Num n) {
        return Num(x + n.x);
    }
};

int main() {
    Num a(5), b(7);
    Num c = a + b;
    cout << c.x;
    return 0;
}

19. Advanced Topics

Advanced

Advanced C++ topics include namespaces for modular programming, modern type casting (static_cast, dynamic_cast, const_cast, reinterpret_cast), function pointers, virtual tables (vtable), memory management internals (stack vs heap), design patterns, macros, and understanding the compilation process.

Example Question: Use function pointers to perform different mathematical operations. 


#include <iostream>
using namespace std;

int add(int a, int b) { return a + b; }
int multiply(int a, int b) { return a * b; }

int main() {
    int (*operation)(int, int);

    operation = add;
    cout << "Add: " << operation(5, 3) << endl;

    operation = multiply;
    cout << "Multiply: " << operation(5, 3) << endl;

    return 0;
}

20. Projects / Real-world Applications

Advanced

Building projects helps consolidate C++ concepts and prepares you for real-world problem solving. Common projects include calculators, student management systems, banking applications, inventory systems, games, and modern C++ applications using STL, smart pointers, and OOP.

Example Question: Build a mini calculator using switch-case. 


#include <iostream>
using namespace std;

int main() {
    char op;
    double a, b;

    cout << "Enter operator (+, -, *, /): ";
    cin >> op;

    cout << "Enter two numbers: ";
    cin >> a >> b;

    switch(op) {
        case '+': cout << a + b; break;
        case '-': cout << a - b; break;
        case '*': cout << a * b; break;
        case '/':
            if(b != 0) cout << a / b;
            else cout << "Division by zero!";
            break;
        default:
            cout << "Invalid operator";
    }
    return 0;
}
