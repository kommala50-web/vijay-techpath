HTML Full Roadmap

Learn HTML from absolute basics to advanced concepts. Click Show Code to view HTML examples.

1. Introduction to HTML

Beginner

HTML defines the structure of a webpage using elements like headings, paragraphs, links, images, and forms. 


<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

2. HTML Document Structure

Beginner

Every HTML page follows a fixed structure using html, head, and body tags. 


<html>
  <head>
    <meta charset="UTF-8">
  </head>
  <body>
    Page content
  </body>
</html>

3. Headings & Paragraphs

Beginner

Headings define importance and paragraphs define readable text content. 


<h1>Main Heading</h1>
<p>This is a paragraph</p>

4. Links & Images

Beginner

Links allow users to navigate between pages using the anchor tag. Images are added using the img tag. The alt attribute improves accessibility and SEO. 


<a href="https://example.com">Visit Example</a>

<img src="image.jpg" alt="Sample image">

5. Lists

Beginner

Lists are used to group related items. Ordered lists use numbers, unordered lists use bullets, and definition lists describe terms. 


<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol>
  <li>Step One</li>
  <li>Step Two</li>
</ol>

6. Tables

Beginner

Tables display data in rows and columns. They are commonly used for reports, schedules, and structured data. 


<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>20</td>
  </tr>
</table>

7. Forms

Beginner

Forms collect user input such as names, emails, and passwords. They are widely used for login, signup, and feedback. 


<form>
  <input type="text" placeholder="Name">
  <input type="email" placeholder="Email">
  <button type="submit">Submit</button>
</form>

8. Input Types

Beginner

HTML provides many input types like text, email, password, number, checkbox, and radio to improve user experience and validation. 


<input type="password">
<input type="number">
<input type="checkbox">

9. Semantic HTML

Intermediate

Semantic tags describe the meaning of content. They improve readability, accessibility, and SEO by clearly defining page sections. 


<header>Header</header>
<main>Main Content</main>
<footer>Footer</footer>

10. Div & Span

Beginner

Div is a block-level container used for layout, while span is an inline container used for styling small pieces of text. 


<div class="box">
  This is a div
</div>

<span>Inline text</span>

11. Semantic HTML

Intermediate

Semantic HTML uses meaningful tags that clearly describe the role of the content inside them. Instead of using generic <div> tags everywhere, semantic elements like <header>, <nav>, <main>, <section>, <article>, and <footer> explain the structure of the page to browsers, search engines, and screen readers. This improves accessibility, SEO, and code readability. 


<header>
  <h1>My Website</h1>
  <nav>
    <a href="#">Home</a>
    <a href="#">About</a>
  </nav>
</header>

<main>
  <section>
    <h2>Blog Posts</h2>
    <article>
      <h3>Post Title</h3>
      <p>This is a blog post.</p>
    </article>
  </section>
</main>

<footer>
  <p>© 2026 My Website</p>
</footer>

12. Tables

Beginner

HTML tables are used to display data in rows and columns. A table is created using the <table> tag, rows using <tr>, headers using <th>, and data cells using <td>. Tables are commonly used for reports, schedules, marks lists, and structured data, but should not be used for page layout. 


<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>21</td>
  </tr>
</table>

13. Forms

Beginner

Forms are used to collect user input such as names, emails, passwords, and feedback. The <form> element wraps input fields, and data is usually sent to a server using the action and method attributes. Forms are essential for login pages, signup pages, and contact forms. 


<form>
  <label>Name:</label>
  <input type="text" />

  <label>Email:</label>
  <input type="email" />

  <button type="submit">Submit</button>
</form>

14. Input Types

Beginner

HTML provides different input types to improve user experience and validation. Common input types include text, password, email, number, checkbox, radio, date, and file. Choosing the correct input type helps browsers validate data automatically and provides better controls on mobile devices. 


<input type="text" placeholder="Username" />
<input type="password" placeholder="Password" />
<input type="number" />
<input type="checkbox" /> Subscribe

15. Audio & Video

Beginner

HTML allows embedding audio and video directly without external plugins. The <audio> and <video> tags support controls like play, pause, and volume. This is commonly used for tutorials, music players, and media content. 


<video controls width="300">
  <source src="video.mp4" type="video/mp4">
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

16. Iframes

Intermediate

Iframes allow embedding another webpage inside the current page. They are commonly used for embedding maps, videos, dashboards, or external tools. Use iframes carefully as they can affect performance and security. 


<iframe
  src="https://example.com"
  width="300"
  height="200">
</iframe>

18. Accessibility

Advanced

Accessibility ensures websites are usable by people with disabilities. Using proper labels, alt text, semantic tags, and ARIA attributes helps screen readers understand content. Accessibility is not optional — it is a professional requirement. 


<button aria-label="Close Menu">X</button>
<img src="logo.png" alt="Company Logo">

19. SEO Basics

Advanced

Search Engine Optimization (SEO) helps webpages rank higher in search results. Proper use of headings, meta tags, alt attributes, semantic HTML, and meaningful content improves visibility and discoverability. 


<h1>HTML Roadmap</h1>
<meta name="description" content="Learn HTML step by step">
<img src="html.png" alt="HTML logo">

20. Mini Project

Advanced

A mini project helps you apply everything you learned. A simple HTML project can include a structured page with header, navigation, content sections, forms, images, and footer. This builds confidence and prepares you for CSS and JavaScript. 


<header>My Website</header>
<main>
  <section>About Me</section>
  <section>Contact Form</section>
</main>
<footer>© 2026</footer>
