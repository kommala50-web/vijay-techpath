Year 01

Build Strong IT & Networking Fundamentals

  • Skills: Python, C/C++, OSI, TCP/IP, DNS, VPN, Firewalls
  • Linux (Kali/Ubuntu), Windows internals, Command line
  • Cybersecurity basics: threats, vulnerabilities, exploits
  • Projects: Linux lab, Wireshark analysis, network simulation
  • Tools: Wireshark, Nmap, VirtualBox
  • Certifications: CompTIA ITF+, Linux Essentials
Year 02

Core Cybersecurity & Ethical Hacking

  • Skills: Pentesting, SQLi, XSS, CSRF, OWASP Top 10
  • Network security: IDS/IPS, MITM, VPNs
  • Cryptography: hashing, encryption, TLS
  • Projects: Pentesting lab, scanning networks, password cracking
  • Tools: Kali, Burp Suite, Metasploit, Hydra
  • Certifications: CEH (beginner), Security+
Year 03

Advanced Pentesting & Defense

  • Skills: Exploit dev, buffer overflow, post-exploitation
  • Incident response, forensics, malware analysis
  • Hardening, patching, vulnerability management
  • Projects: Full PenTest, keylogger, CTF events
  • Tools: Metasploit, SQLmap, Nikto, Hashcat
  • Certifications: CEH Practical, PenTest+, eJPT
Year 04

Specialization & Job Preparation

  • Specializations: Web pentesting, cloud security, red/blue teaming
  • Advanced tools: Splunk, ELK, threat intel, cloud security
  • Cyber laws, GDPR, ISO compliance
  • Projects: Full-scale pentest, server hardening, open-source
  • Certifications: OSCP, eCPPT, CySA+, CISSP
  • Career Prep: Resume, portfolio, mock interviews
